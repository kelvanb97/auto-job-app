# @aja-app/supabase

Used to generate types from supabase

```console
$ pnpm run typegen:local
$ pnpm run typegen:production
```

## Soft Links

`config.toml` file is created via soft link
from the root [apps/supabase/config.toml](../../../apps/supabase/config.toml).

They are both set to readonly by [.vscode/settings](../../../.vscode/settings.json)
